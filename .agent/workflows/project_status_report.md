---
description: Scans the project structure, reads key configuration files, and generates a comprehensive status report on the project's purpose, tech stack, and recent state.
---

1. List the root directory to understand the general project structure.
2. Read `README.md` to identify the project's purpose, features, and documentation.
3. Read `package.json` to identify dependencies, scripts, and the technology stack.
4. List the contents of `src` and `src/app` to understand the source code organization.
5. Summarize the findings into a report covering:
   - Project Name & Purpose
   - Key Features
   - Tech Stack (Frameworks, Libraries)
   - Current File Structure State
